# CHANGELOG

## Назначение

Этот документ хранит историю релизно значимых изменений.

Формат можно вести по принципам Keep a Changelog:

- `Added` - добавлено.
- `Changed` - изменено.
- `Deprecated` - будет удалено.
- `Removed` - удалено.
- `Fixed` - исправлено.
- `Security` - изменения безопасности.
